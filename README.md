## 🚀 快速启动

### 一键启动所有服务
```bash
bash start_all.sh
```

启动后可访问：
- 🔌 **API服务**: http://127.0.0.1:8000/status
- 📊 **运营报告**: http://127.0.0.1:8080/report
- 📱 **微信集成**: http://127.0.0.1:8088/report
- 🌐 **报告中心**: http://127.0.0.1:8089

### 系统自检
```bash
python3 system_check.py
```

### 生成运营报告
```bash
python3 generate_operation_report.py
```
