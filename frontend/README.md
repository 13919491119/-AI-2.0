# 玄机AI智能预测系统前端

## 功能概览

- 双色球预测（开奖时间、期次、历法、预测模式等）
- 八字排盘（阴阳历、出生年月日时）
- 占卜断事（事项、起盘时间、模式）
- 八字起名（性别、历法、时间、姓氏）
- 系统仪表盘（核心状态展示）

## 技术栈

- React 18
- Ant Design 5
- Vite 5
- axios

## 启动开发

```bash
cd frontend
npm install
npm run dev
```

## API对接

- 默认代理到 <http://localhost:8000/api>
- 可根据实际后端API调整

---
本前端为AI系统自动生成，支持自定义扩展。
